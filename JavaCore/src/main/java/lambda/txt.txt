Predicate<T>   boolean test(T t)
используется методом removeIf для указания условия по которому можно удалить обьект
делает выборку по определенному условию


Supplier<T>   T get();
поставляет обьект, когда вызывается метод
можно создать новый обьект

Consumer<T>   void accept(T t)
функциональность Supplier и Consumer противоположны
модифицирует обьект в коллекции по заданным параметрам
можно перебрать коллекцию по типо цикла forEach

Function<T, R>  ->  R apply(T t)
для определения среднего арифметического значения
передаем студента -> на выхоже получаем double из среднего арифметического его оценок
